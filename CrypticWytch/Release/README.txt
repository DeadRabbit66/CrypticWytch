🔐 НАЧАЛО РАБОТЫ:

Нажмите «Gen & Copy» для создания ключа, копирования его в буфер и входа в программу. 
Этот ключ нужно передать собесеседнику, либо сохранить где-то у себя для самостоятельной работы.

Нажмите «Import Key», если в буфере уже есть скопированный ключ.

⚠️ ВНИМАНИЕ! 
Ключ, используемый для входа, шифрует и дешифрует всю информацию в текущем сеансе. 
После закрытия программы он БЕЗВОЗВРАТНО ТЕРЯЕТСЯ. 
Если ключ от сеанса не сохранён отдельно — дешифровка информации из закрытого сеанса будет невозможна.

───────────────────────────────────────────

✨ ПОСЛЕ ВХОДА В ПРОГРАММУ:

🔒 Шифрование: Введите текст в окно → нажмите «Encrypt & Copy» → зашифрованный текст скопируется в буфер.

🔓 Дешифровка: Нажмите «Paste & Decrypt» → текст из буфера расшифруется и появится в окне.

🔄 Сброс сеанса: Нажмите «Reset Session Key» — текущий ключ будет утерян, программа вернётся на экран генерации.

───────────────────────────────────────────

⚙️ ТЕХНИЧЕСКИЕ ДЕТАЛИ:

Алгоритм: AES-256-GCM (аутентифицированное шифрование)
Формат ключа: Base64 (44 символа / 32 байта)

---

GETTING STARTED:
Press «Gen & Copy» to generate a key, copy it to clipboard, and enter the program.
Share this key with your recipient or save it for later.

Press «Import Key» if you already have a key in your clipboard.

⚠️ WARNING! The key is lost when the program closes.
    Save it separately if you need to decrypt data later.

───────────────────────────────────────────────────────────────

✨ AFTER ENTERING THE PROGRAM:

🔒 Encrypt:     Enter text → «Encrypt & Copy» → encrypted text in clipboard
🔓 Decrypt:     Press «Paste & Decrypt» → decrypted text appears in the window
🔄 Reset session: «Reset Session Key» — return to key generation

───────────────────────────────────────────────────────────────

⚙️ TECHNICAL DETAILS:
Algorithm: AES-256-GCM | Key format: Base64 (44 characters)
